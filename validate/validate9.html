Validate Form Example

Javascript, JQuery, JSON, CSS

A basic, preliminary form validation routine using Javascript and JQuery. Of course, a more thorough validation of the submitted form would be done by the server-side script (not performed in this example). This small, light weight routine perfoms the following checks: 1) Required Fields. Fields that are selected as required are checked to make sure a value is entered. 2) Field Length. The entered fields's minimun and maximun length can be checked. It can also check that a checkbox has a minimun and/or maximun number of boxes checked. 3) Field Format. Regular expressions are used to check the format of fields such as dates, emails, alphanumeric etc. Other checks can easily be added.

In this example: 1) All fields are Required except the phone field, so they will be checked that a value is entered. Although the phone field is not required, it will be checked if a value is entered. 2) The value entered in the Name field must have a length of 4 to 30 characters. Length checks are not done on the Email, Birthday and Phone fields since the regular expresion check will be satisfactory. 3) Regular expersion checks are done on the Name, Email, Birthday and Phone fields. 4) A length check is done on the Colors checkbox: Exactly two items must be checked.

It is easy to set up fields to be checked. Just add the new field to a JSON array of fields to be checked. Custom check routines can be added. Here is the array for the Name field:
"name": [
{ 'info':{"fieldname" : 'Name', "fieldid" : 'name', "dataType" : 'words' , "errormessage" : ''}},
{ 'required':{"required" : 'yes', "errormessage" : ': required field' }},
{ 'regex':{"regex" : /^[a-zA-Z0-9 '.]*$/ , "errormessage" : ': format error' }},
{"cklength" :{"cklength" : '', "minlength" : 1, "maxlength" : 30, "errormessage" : ': length error' }}
],
